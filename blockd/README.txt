 - - - - - - - - - - - - - - - Blockd - - - - - - - - - - - - - - -
    A voxel survival game blk_based off of Bloxd.io a web voxel game.
Blockd is not to ba a one to one recreation, instead it strives to be
similar, but not the same.
    Blockd is still in development, as such there are no
guaranties that it will work as intended, or that it will not corrupt
game data. If you notice any issues please report them on GitHub,
you will find the web address in the contact section below.
