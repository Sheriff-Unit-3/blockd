# Blockd
An open world survival game built for the Luanti Game Engine. With some similarities to various voxel games.

## Features
* Open world to explore and conquer, with mines, caves, forrests, plains, tundras, and deserts. 
* 6 types of wood, 15 types of wool and dye, various plants and crops, and 6 types of ore.  
* Deep mining with some ores not starting until -25000 y-level.  
* Plus active development that is adding more features all the time.  

## Goals
These are some features that are planned to be released in the future.  
[ ] Furniture
[ ] Improved farming mechanics  
[ ] Uses for Emeralds and Lapis  
[ ] More ores (silver, copper, tin, etc..)  
[ ] Hunger  
[ ] Mobs  
[ ] More biomes
